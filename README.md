# 问道MMORPG - 启动说明

## 快速开始

1. 双击 `start.bat` 启动游戏服务器
2. 服务器启动后，浏览器会自动打开游戏页面
3. 如果没有自动打开，手动访问：`http://localhost:3000`

## 技术说明

- 服务端：Node.js + Express + WebSocket
- 客户端：HTML5 + CSS3 + JavaScript
- 数据库：JSON文件（无需安装数据库）
- 端口：3000

## 关闭游戏

1. 关闭浏览器中的游戏页面
2. 在服务器窗口按 `Ctrl+C` 停止服务器
3. 或者直接关闭服务器窗口

## 问题排查

如果无法访问游戏：
1. 确认服务器已启动（看到"问道MMORPG服务器启动成功！"）
2. 确认端口3000未被占用
3. 尝试访问 `http://127.0.0.1:3000`
